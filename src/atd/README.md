### Generic function based in std.

It is generic code. The only requirement to work with microcontrollers is:
1) Can't throw exceptions.
2) Can't use memory allocations.

Tested with: avr-gcc 7.4.0.

### Funciones genéricas que se basan en std. 

Es código genérico que se puede compilar también con gcc. El único requisito impuesto
para que funcionen con los microcontroladores es que no lancen excepciones ni usen
memoria dinámica.


Probado con: avr-gcc 7.4.0
